A GCM using the 6-hour forecast modelÂ¶

The final step to make a GCM is to run the predictor repeatedly, using the output from the previous run as the input to the next run. This works, but if the model is left completely free to run it does not have reliable diurnal and annual cycles. I got better results by editing the top-of-atmosphere insolation field in each state (which depends only on time and date) to make sure it was always correct - effectively forcing the model with the TOA radiation pattern.
